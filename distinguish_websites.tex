\section{引言}
近年来，虚假信息在互联网上愈发猖獗。舰C吧70W事件\cite{genC15}和血友病吧事件\cite{blood16}，使得互联网巨头百度名声狼藉。
无尽的虚假诈骗信息，欺骗着无辜的网友。
显然，如何鉴别一个网站的信息是否真实可靠，已经成为一个迫在眉睫的问题。
本文总结了鉴别网络信息的一般方法，并列举了若干个著名可靠网站，以及若干个不应被信任的典型网站。也算是尽了笔者的一份绵薄之力。

\section{鉴别网络信息的方法与技巧}
\subsection{批判性思维的力量}
世间本无完美的文章。即便是顶级的学报和会议记录，都时常有小错误被发现。这也是批判性思维之所以如此重要的原因。
在阅读文献时，应当审慎地确保其内容能够说服自己。通常，这涉及到下面两个问题：
\begin{itemize}
\item 文章的逻辑是否合理？有无纰漏？
\item 文章的论据是否都有真实的引证来源？
\end{itemize}
对于第一个问题，如果内容在自己的知识范畴内，只要有足够的耐心和细心，应该是可以充分验证逻辑的合理性。
如果材料内容过于专业，可以选择先搁在一旁。尽量不要接受任何未经逻辑证实的信息。

对于第二个问题，读者应当仔细地验证每一条引证的真实性。
一般来说，可靠的引证来源来自于权威新闻社（比如路透社、法新社、美联社、合众社、纽约时报等），大型组织的官方网站（比如国际奥委会、苹果公司等），以及著名高校的官方网站。
这些网站给出的一般都是{\bf 一手资讯}，其他转述的网站有些可信度也很高（比如维基百科），但产生的都是二手资讯，有轻微曲解原意的风险。
验证引证来源的真实性非常重要！事实上，编造、歪曲论据是欺骗的最常见手段之一。

值得注意的是，目前很多网站上的文章都没有或者只有很少的论证引用来源，导致论据的验证工作往往难以进行。那么这样的信息我们是否该相信呢？
一个稍微严谨的观点是，应当拒绝去读这样的文章。首先，给出充分的引证来源应当是作者应尽的义务。
一篇没有充足引证来源的文章，往往是劣质的。你不值得在劣质的文章上浪费宝贵的时间。
这也是维基百科有别于百度百科的最大区别之一。在维基百科中，尤其是社会科学的文章中，一个论据往往需要多方引证，以确保在论述上是客观的。比如英文维基百科在伊斯兰国一文\cite{isis15}中，就给出了超过七百处引证，其中包括来自四大社、纽约时报、华尔街时报、华盛顿邮报、NBC等著名新闻媒体的报道，从最大限度上保证了观点的平衡。
反观百度百科，在引用论证上明显做得不够，很多只是依靠单一的消息来源，客观性和可信度都不高。

\subsection{真金不怕火炼}
一篇好的文章应当是被人们反复检查过的，这样可以让出错的概率降到最低。这也是为什么读文献时，要优先读被引用次数多的文章。
同样的道理，一篇优秀的文章也是需要不断的改进和维护的。因此一个做维护工作的团队就必不可少。
在维基百科中，用户可以在每一篇文章的讨论页上发表自己的批判性见解，或提出改进建议，或与其他用户进行讨论。这才是严谨的编辑应有的氛围！
而在百度百科中，笔者并没有看到任何可供编辑讨论的平台。

\subsection{客观公正的网站应当拒绝广告}
接收太多的广告势必会影响网站内容的客观性和中立性。这点也是维基百科一直以来强烈抵制广告的最主要原因\cite{wiki16}。
非常遗憾的是，很多所谓大门户网站（比如新浪、腾讯、百度、网易等），各种广告充斥着网页。有些网站甚至容许广告软文的存在。
这些都不利于获取客观中立的信息！


\section{可靠网站}
\begin{enumerate}
\item {\bf 知名高校的官方网站} 由于这类网站发布信息的正确性与高校的声誉息息相关，它们通常都会被专业人员很谨慎地维护着，所以具有很高的可信度。
\item {\bf 维基百科} \cite{wiki} 详实的论证，积极的维护，及时的讯息，纯净无广告，其准确性无需多言。不过值得注意的是，较为专业的技术内容，在维基百科上往往很少。这时只有求助于专业文献或者大学网站上的相关课程讲义了。
\item {\bf 学术网站} 目前已有许多优秀的存放学术文献的专门网站，它们都是免费开放的，比如nLab \cite{nlab}, dblp \cite{dblp}, arXiv \cite{arxiv}, iacr \cite{iacr}, eccc \cite{eccc}等等。除此之外，Google Scholar \cite{scholar}可以用来方便地检索它们。事实上，很多时候Google Scholar的引用排名比SCI的更为客观。关于这点，Yvo Desmedt有过论述\cite{YD15}。Rosario Gennaro也同意这一观点\cite{RG15}。
\item {\bf 类Quora \cite{quora}的问答网站} 比如stackoverflow \cite{so}, stackexchange \cite{se}, mathoverflow \cite{mo}等。这类网站在用户对答案的互动交流上做得很好，通过对答案进行投票和反馈，可以很容易地让优秀答案脱颖而出。
Quora是相对大众向的问答网站，其他几个专注于具体的学科。

中文的优秀问答网站不多，或许知乎\cite{zhihu}可以算一个。
不过知乎在问题的管理上做得不如Quora它们那么出色：对无意义的回答缺少处理手段；对无意义提问缺少处理手段等等。
\item {\bf 其他优秀的wiki网站} 比如ニコニコ大百科\cite{nico}，萌娘百科\cite{moe}等。萌娘上有些休闲、吐槽的内容，不过应该不影响整体的使用。 
\end{enumerate}

\section{反面教材}
\begin{enumerate}
\item {\bf 百度产品} 如引言所述，接连不断的丑闻已经让百度声名狼藉。百度搜索的排名会被广告所影响；百度贴吧几乎被推广广告所淹没；百度百科的维护乏善可陈，还有抄袭其他信息来源而不给引用的嫌疑\cite{baidu1600}。出于严谨的态度，应当拒绝百度的任何产品！

\item {\bf 一票所谓门户大网站}
这些所谓的大网站充斥着严重影响中立性的广告，还往往提供新闻服务。不幸的是，这些新闻发表的门槛非常低，很多稿件发布后都有大量的错字，消息引证更是不被重视，甚至会有擦边球、欺骗性软广告出现在醒目的推广位置。无数的事实迫使我们拒绝相信它们传播的信息内容！
\end{enumerate}
